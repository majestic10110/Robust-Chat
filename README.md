Robust Chat is a chat program that utilizes the LiNK500 TNC, SCSTracker(should be functional), TEENSY RPR TNC. 

Prequisites: Setup a folder called Robust Chat anywhere; Ensure the following files positions.json, freqs.json, radios.json , messages_v1.json , settings.json and link_graph.json live in the same folder as you put the Robust_Chat_(version).exe
